# 🚀 Agile Program Dashboards

A Power BI portfolio that showcases real-world Agile delivery insights. This repo includes two professional-grade dashboards built to help teams, product owners, and program managers track sprint execution, milestone progress, risk distribution, and cross-team coordination.

---

## 📁 What’s Inside

### 1️⃣ [Agile Metrics Dashboard](./Agile-Metrics-Dashboard)

Tracks delivery performance across sprints:

- 📈 Sprint Velocity (story points trend)
- 🐞 Bug Rate (bugs vs stories delivered)
- ⏱️ Cycle Time (average completion time)
- 🔥 Burnup (planned vs delivered effort)
- 🗂️ Feature Status Tracker

🎯 Built for: Scrum Masters, Agile Coaches, Delivery Managers

---

### 2️⃣ [Program Delivery Insights](./Program-Delivery-Insights)

Highlights cross-team and milestone-level progress:

- 🧱 Team vs Status Matrix
- 🍩 Milestone Completion Donut
- 📈 Feature Burnup Chart (Planned vs Delivered)
- 🔥 Risk Heatmap (High/Med/Low by team)
- 🔗 Dependency Table (cross-team blockers)

🎯 Built for: Program Leads, PMOs, Stakeholders

---

## 🧠 Tools & Techniques

- **Power BI Desktop** for dashboard creation
- **DAX expressions** for custom KPIs, flags, and logic
- **Matrix, Donut, Column, KPI, Heatmap visuals**
- **Mock datasets** simulating real-world Agile delivery data
- Structured to support GitHub-based portfolios & README documentation

---

## 💼 Use Cases

- Demonstrate delivery performance in retros or sprint reviews  
- Share release readiness status with stakeholders  
- Showcase your Power BI, DAX, and Agile dashboarding skills in interviews  
- Build trust through transparent feature tracking and risk visualization
